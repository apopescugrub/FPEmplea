Fuente Open Sans para parrafos.

Fuente Playfair Display negrita para parrafos importantes.

Font Size

Hay 4 tamaños de letra, se pueden usar añadiendo una de las clases:


.f1{font-size:1em}

Hola Mundo!

.f2{font-size:1.2em}

Hola Mundo!

.f3{font-size:1.9em}

Hola Mundo!

.f4{font-size:2.5em}

Hola Mundo!
