Botones

Para el formato de cualquier boton hace falta añadir la clase .btn

Para los distintos tamaños de botones hace falta añadir la clase .x1 .x2 .x3 class="btn x1"


<button type="button" class="btn btn1 x1">Hola</button>

<button type="button" class="btn btn1 x2">Hola</button>

<button type="button" class="btn btn1 x3">Hola</button>

Para los botones desabilitados se usará la clase disabled.


<button type="button" class="btn disabled">Hola</button>

Se pueden combinar las clases anterior con la clase disabled.


<button type="button" class="btn btn1 x1 disabled">Hola</button>

<button type="button" class="btn btn1 x2 disabled">Hola</button>

<button type="button" class="btn btn1 x3 disabled">Hola</button>

Segundo Boton

Para cambiar de tipo de boton se tiene que añadir la clase .btn1  , .btn2  , .btn3 que se pueden combinar con cualquiera de las clase anterior pero no entre ellas.

Segundo boton tamaño 1.

<button type="button" class="btn btn2 x1">Hola</button>


Segundo boton tamaño 2.

<button type="button" class="btn btn2 x2">Hola</button>


Segundo boton tamaño 3.

<button type="button" class="btn btn2 x3">Hola</button>


Segundo boton deshabilitado.


<button type="button" class="btn btn2 x1 disabled">Hola</button>

<button type="button" class="btn btn2 x2 disabled">Hola</button>

<button type="button" class="btn btn2 x3 disabled" >Hola</button>
