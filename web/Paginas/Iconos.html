Uso Material Design Icons
( Google)

Se debe a単adir la clase .material-icons seguida de la clase correspondiente al tama単o.
Y escribir el nombre del icono dentro de la etiqueta <i>
.material-icons.md-18 { font-size: 18px; }
.material-icons.md-24 { font-size: 24px; }
.material-icons.md-36 { font-size: 36px; }
.material-icons.md-48 { font-size: 48px; }

Ej:

sentiment_satisfied<i class="material-icons md-18">sentiment_satisfied</i>
sentiment_satisfied<i class="material-icons md-24">sentiment_satisfied</i>
sentiment_satisfied<i class="material-icons md-36">sentiment_satisfied</i>
sentiment_satisfied<i class="material-icons md-48">sentiment_satisfied</i>
